Introduction to Wikipedia

Welcome to Wikipedia! Anyone can edit almost every page, and millions already have.

This page takes you through a set of tutorials aimed at complete newcomers interested in contributing. It covers all the basics needed to navigate, comment on, and contribute to Wikipedia articles. Each tutorial takes only a few minutes, so you could become a proficient Wikipedian in no time!
